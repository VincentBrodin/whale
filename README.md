## Whale 🐋

**Whale** is a lightweight, pluggable Terminal UI for Go.  
Named after the Swedish word **“val”** (*choice* / *whale*), it helps you build interactive prompts with lists, confirmations, fuzzy search, and more—effortlessly.

- [X] List
- [ ] List (Searchable)
- [ ] Multi-choice list
- [ ] Confirmation
- [ ] Text
